Run `./lf-kitty`, no installation is required.

**Prerequisites**

+ A terminal that implements the [kitty graphics protocol](https://sw.kovidgoyal.net/kitty/graphics-protocol/)
+ `graphicsmagick` for svg and gif previews
+ `gs` for PDF previews
